# Task Tracker Application

This is a simple Task Management Web Application developed using HTML , CSS and JavaScript.

## Technology Used

### Frontend
- HTML
- CSS3
- JavaScript

### Storage
- Browser Local Storage

## Features Completed
- Add task
- View task list
- Mark task as completed
- Delete task

## How to Run
1. Download the project zip.
2. Extract the files.
4. Open the `index.html` file in your browser.

## Challenges Faced
- Managing task states such as complete and delete required careful handling of task data.
- Designing a simple and intutive interface.

## Improvements Planned
- Store Tasks in a database and sync across multiple devices.
- Allow task Reordering.
- UI Enhancement.
- Add date and time to tasks.
